# Doces Gourmet - Loja Online

Este é um site de demonstração para a loja de doces caseiros **Doces Gourmet**, criado por Graciele Rocha.  

## 📌 Funcionalidades

- Página principal com produtos à venda (brownies, cookies, docinhos)
- Página de sabores com todos os tipos de doces disponíveis
- Página sobre a empresa e equipe
- Carrinho de compras com adição, remoção e finalização de pedidos
- Layout responsivo e atraente
- Todos os produtos e informações reais da loja

## 🛠 Tecnologias

- HTML5
- CSS3
- JavaScript (localStorage para carrinho)
